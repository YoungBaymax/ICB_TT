Detailed Description

Algorithms and functions for 9DOF fusion.

Uses data from accelerometers, gyroscopes and magnetometers to do sensor fusion and obtain orientation of the device in a quaternion representation.

Author
Sebastian Sepulveda

def sensorFusion.SensorFusion.update9DOF (   self,
  gyroscope,
  accelerometer,
  magnetometer 
)

Update the orientatation with sensors data.

Sensor fusion algorithm ported by Fabio Varesano to Python from the Sebastian Madgwick 9DOF fusion.

Data must be inputed as:

  • Accelerations in g (gravities)
  • Gyroscope in radians per second (rad/sec)
  • Magnetometer in Gauss (Gs)
Parameters
selfThe object pointer
gyroscopeGyroscope data as [X,Y,Z] in rad/sec
accelerometerAccelerometer data as [X,Y,Z] in g (gravities)
magnetometerMagnetometer data as [X,Y,Z] in Gs
